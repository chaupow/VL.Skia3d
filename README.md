# VL.Skia3d

A simple 3d addon for VL.Skia

With support for the following primitives:
* Points
* Line
* Circle
* Rectangle
* PolyLine
* Box
* Axis
* Grid
* Billboard Text

Plus a Camera (Orbit) and Transform nodes to plug below the primitives (VL.Skia style).
